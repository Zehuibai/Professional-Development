CheatSheet -Basic Plots and Customization and Maps

Command Syntax Description Example
Scatter Plot qplot(data=data,feature1,feature2) It plots the scatter plot between two continuous variables feature1 and feature2, helps us to find the relationship between the two variables. qplot(data=mtcars,mpg,wt)
Scatter Plot - Customization ggplot(data,aes(x=feature1,y=feature2))+geom_point(shape=1) It plots the scatter plot between two continuous variables feature1 and feature2, helps us to find the relationship between the two variables. We can do more customization in ggplot. shape property in geom_point changes the shape of the points in the scatter plot. ggplot(mtcars,aes(x=wt,y=mpg))+geom_point(shape=1)
Line Plots ggplot(data,aes(x=Date-Feature,y=feature2))+geom_line() Line plots help you visualize time series data. To create a line plot, you must first use the ggplot() function to specify variables for the x and y-axis. Then, to create the line, you add the geom_line() function. By default, the x and y-axis labels will be the name of the variables you specified when creating the plot. ggplot(eustock,aes(x=Date,y=SMI))+geom_line()
Boxplot ggplot(data,aes(x=label,y=feature1))+geom_boxplot() To run the ggplot method with our data frame as input, setting the x axis to display the labels, and the y axis to display the range of numbers. To get a box plot as output, we need to add the “geom_boxplot” method at the end. ggplot(mtcars,aes(x=cyl,y=mpeg))+geom_boxplot()
